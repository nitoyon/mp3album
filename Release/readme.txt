【概要】
複数の MP3 ファイルを無圧縮 zip 形式で固めることで、ＣＤアルバムをファ
イルとして保存することができます。この形式は、MP3 プレイヤーでそのまま
聞くことができます。このソフトは zip.mp3 ファイルを簡単に作成できます。

zip.mp3 でアルバムを保存しておけば、ＣＤの枚数単位でののランダム再生が
可能になります。ＣＤプレーヤーのチェンジャー機能が不要になってちょっと
幸せじゃないでしょうか・・・。


【インストール】
お好きなフォルダに解凍するだけでＯＫです。


【アンインストール】
ファイルごと削除して下さい。
設定は実行ファイルと同じフォルダに mp3album.ini として保存されます。
レジストリやシステムフォルダにファイルは保存しません。


【簡単な使い方】
zip.mp3 化したい MP3 をエクスプローラから 「ファイル一覧」 にドラッグ＆
ドロップしてください。「曲名」と書いてある部分をクリックするとソートさ
れます。また、「↑」「↓」ボタンを押すことによって任意の順番に入れ替え
ることができます。
作成したいアーカイブ名を 「出力 zip ファイル」 の欄に入力して、「作成」
ボタンを押せば、作成されます。そのまんまですね。


【詳細】
■ メインウインドウ
・出力 zip ファイル
    zip ファイルを保存するファイルのパスを入力してください。

    フルパスを入力しない場合は、ファイル一覧の先頭のファイルがあるフォル
    ダに作成します。ファイル名を指定しないと album.zip.mp3 という名前
    で作成します。これらのデフォルトの動作は 「設定」 で変更することが
    できます。

・ファイル一覧
    まとめたいファイルの一覧です。
    ドラッグ＆ドロップするか Add ボタンを押すことで追加できます。
    MP3プレーヤー で聞いたときに、この順番でるように圧縮します。

・フルパス
    チェックするとファイル一覧をフルパスで表示します。
    フルパスの場合に、「曲名」バーをクリックするとフルパスの名前でソートします。
・MP3以外は追加しない
    チェックしておくと、MP3 以外のファイルをドラッグ＆ドロップしてもリストには追
    加しません。

・Add
    リストにファイルを追加します。
    ファイル選択ダイアログでは Ctrl を押しながらクリックすることで複数選択可能で
    す。

・Del
    選択したファイルを一覧から削除します。Del キーを押しても削除できます。
    複数選択した状態でこのボタンを押すと、一度にたくさんの曲をリストから削除で
    きます。

・↑ ↓
    選択したファイルを上下に移動します。
    Alt + ↑, Alt + ↓ キーでも移動可能です。

■ 設定ダイアログ
・デフォルトのフォルダ
    メインウインドウの「出力 zip ファイル」をフルパスで入力しなかった場合、
    どこに zip ファイルを作成するかを設定する。

・デフォルトのファイル名
    メインウインドウの「出力 zip ファイル」でファイル名を入力しなかった場合の
    ファイル名を指定する。

・デフォルトの拡張子
    メインウインドウの「出力 zip ファイル」で拡張子を入力しなかった場合の拡張
    子を指定する。

    ※ 補足
    例えば
        デフォルトのフォルダ  : C:\tmp\
        デフォルトのファイル名: album
        デフォルトの拡張子    : .zip
    だったとする。

    このときには
        何も入力せずに作成  → C:\tmp\album.zip
        D:\ と入力して作成  → D:\album.zip
        hoge と入力して作成 → C:\tmp\hoge.zip
    となる。

・圧縮後にログを表示する
    チェックしておくと圧縮作業が終了後、ログを表示する。
    「失敗時のみ」もチェックしておくと、成功した場合はログを表示しない。

・同名の出力ファイルがある場合は上書きの確認をする
    そのまま。

【既知の不具合】
このソフトで圧縮したファイルを解凍すると、ファイルの作成日時が解凍した時刻になる。


【ＦＡＱ - よくある質問】

Q. 圧縮して作成したファイルは、どうすれば元に戻る？ 

  作成したファイルは、単なる zip アーカイブです。そのため、Lhaplus や Lhaca な
  どの解凍ユーティリティーを使えば解凍できます。

Q. 再生できないファイルがある 

  zip.mp3 には、zip 圧縮するためのファイル情報が記載されているので、mp3 プレイ
  ヤーがその部分を異常なものとして認識した場合、再生できないことがあるかもしれ
  ません。

  もし「他の圧縮ソフトもうまくいくのに、mp3アルバムメーカーなら無理！」という
  状況であればお知らせ下さい。

Q. アーカイブファイルを作成できないんだけど 

  まず、MP3アルバムメーカーのバージョンをご確認下さい。最新版は、1.1.0 となっ
  ております。

  最新版の場合は、具体的にどのような症状かを具体的にお知らせ下さい。そのために
  たとえば、次のようなことを試し下さいませんでしょうか。

  特定のファイル（名前やパス、サイズなど）を圧縮しようとしたときのみの症状では
  ないか（１つだけのファイルから作成してみる、など）
  どのような状況でファイルが作成できないか（エラーダイアログが出る、など）
  MP3アルバムメーカーの特定の設定に依存していないか

  以上、ご面倒ですが詳細を調査の上、原因が分かりましたらご連絡頂けると助かりま
  す。

Q. 他にも質問が。 

  サポート掲示板 (http://www.nitoyon.com/nitobbs/nitobbs.cgi?program) もしくは
  メールでどうぞ。
